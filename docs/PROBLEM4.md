## 기능 목록

### 알파벳을 뒤집어서 변환하는 기능
- [ ] 알파벳이 대문자라면 대문자 중에서 반대로 변환하는 기능
- [ ] 알파벳이 소문자라면 소문자 중에서 반대로 변환하는 기능
- [ ] 위 두 기능을 모아서 차례대로 알파벳을 변환하는 기능
- [ ] 바뀐 알파벳을 새로운 문자열에 반환하는 함수
