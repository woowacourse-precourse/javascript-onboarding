## 🚀 기능 목록

1. 입력받은 문자를 하나씩 변환하기 위해 배열 생성

2. 청개구리 사전을 참고해 문자 변경 `changeChar`
    -문자가 대문자인지 소문자인지 비교 (아스키 코드)
        - 대문자 (65 - 90)
        - 소문자 (97 - 122)
    - ex. 67이면 65부터 몇번째인지 확인후 뒤에서 순서 넣기.
      => 90 - (67-65)
      => 122 - (100-97) // 아스키 코드 100
    - 아스키 코드를 다시 문자로 변경 후에 result에 추가한다.
    
### 실행 결과 예시

| word | result |
| --- | --- |
| "I love you" | "R olev blf" |
